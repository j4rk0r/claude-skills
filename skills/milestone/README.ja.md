milestone v2は**2階層キャッシュ**を持つ永続的な開発トラッカーです。コンパクトなメモリスナップショット（~100トークン、自動ロード）で即時ステータスを確認し、権威ファイルで完全な履歴を保持します。サブタスクを`[simple]`または`[complex]`に分類し、複雑な作業の実行前にプランを要求します——同じファイルへの6+回の反復編集という高コストな試行錯誤サイクルを防止します。

## インストール

```bash
npx skills add j4rk0r/claude-skills@milestone --yes --global
```

## コマンド

| フェーズ | コマンド | 説明 |
|----------|----------|------|
| 発見 | `/milestone` | 全マイルストーンをステータスと進捗とともに一覧表示 |
| 発見 | `/milestone <名前>` | コンテキストをロード（ファジーマッチ） |
| 計画 | `/milestone init <名前>` | サブタスク提案付きで新規作成 |
| 実行 | `/milestone start <名前>` | コンパクトコンテキスト付きの新しいターミナルを開く |
| 実行 | `/milestone done <名前> <タスク>` | サブタスクを完了としてマーク |
| レビュー | `/milestone update <名前>` | 作業セッション後の一括更新 |

## 主要機能

- **2階層キャッシュ** — メモリスナップショット（~100 tok）で読み取り、権威ファイルで履歴。毎回完全なファイルを読むより99%安価。
- **複雑度分類** — `[simple]` vs `[complex]`。複雑なタスクはプランが存在するまで**ブロック**。
- **トークン効率ルール** — 同じファイルへの3+変更 → 単一Write（反復Editより10倍安価）。
- **新セッション** — `/milestone start`がコンパクトコンテキスト付きで新ターミナルに`claude`を開く。
- **12のNEVERルール** — スプリットブレイン防止、古いスナップショット、編集アンチパターン。
