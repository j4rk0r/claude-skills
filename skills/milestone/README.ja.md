> **3つの会話で機能を完成させました。4つ目の会話はゼロから始まります。コンテキストが引き継がれないからです。**

milestone は、プロジェクト内にmarkdownファイルとして完全なコンテキストを保存する永続的な開発トラッカーです。各マイルストーンは自己完結型のカプセルです：目標、ステータス付きサブタスク、アーキテクチャの決定、コード参照、そして何をしたか・なぜしたかのログ。任意の会話でロードし、中断したところから正確に再開できます。

## インストール

```bash
npx skills add j4rk0r/claude-skills@milestone --yes --global
```

## コマンド

| コマンド | 説明 |
|----------|------|
| `/milestone` | すべてのマイルストーンをステータス、進捗、クイックロードリンクと共に一覧表示 |
| `/milestone <名前>` | マイルストーンの完全なコンテキストをロード（あいまい一致） |
| `/milestone init <名前>` | 目標とサブタスク付きの新しいマイルストーンを作成 |
| `/milestone add <名前> <内容>` | サブタスク、決定、メモ、または参照を追加 |
| `/milestone done <名前> <サブタスク>` | サブタスクを完了としてマーク |
| `/milestone update <名前>` | 作業セッション後にコンテキストを一括更新 |

## 主な特徴

- **会話間で永続化** — ファイルは `.milestones/` に保存され、どのセッションでも存続
- **自己完結型コンテキスト** — 各ファイルに作業再開に必要なすべてが含まれる
- **プランニングツール検出** — インストール済みのプランニングスキルを自動検出し、結果の統合を提案
- **自動ステータス** — サブタスクのチェックボックスからステータスを自動再計算
- **あいまい一致** — "dash"と入力して"dashboard-propietario"をロード
- **追記のみのコンテキストログ** — 何が起きたか、なぜかの逆時系列記録
- **グローバルスキル、ローカルデータ** — 一度インストールし、プロジェクト固有のデータを作成
