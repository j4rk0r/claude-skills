# Execution Pipelines by Project Type

Ordered execution sequences. Mark each step ✅ (installed), ❌ (not installed), ⚡ (Claude built-in).

## Web Project
brainstorming → planning → design system → copywriting → SEO → implementation → animation/interactions → accessibility audit → performance → analytics → testing → verification → commit

## Product Launch
pricing strategy → brainstorming → planning → landing page design → copywriting → SEO → implementation → analytics → email marketing → A/B testing → verification

## API Development
requirements → architecture → implementation → documentation → security audit → rate limiting → monitoring → SDK generation → testing → verification → commit

## Mobile App
brainstorming → planning → design → implementation → push notifications → analytics → crash reporting → ASO (app store optimization) → testing → verification → commit

## E-commerce
brainstorming → planning → design → product catalog → payments → tax compliance → shipping → fraud detection → copywriting → SEO → analytics → testing → verification

## Content Platform
brainstorming → planning → content strategy → editorial workflow → design → implementation → localization → search optimization → analytics → testing → verification

## Bug Fix
systematic debugging → fix → testing → verification → commit

## Code Ready to Ship
verification → testing → review → commit → PR

## Documentation
planning → writing → writing quality review → comment analysis → verification

## Refactoring
planning → implementation → testing → review → verification → commit
