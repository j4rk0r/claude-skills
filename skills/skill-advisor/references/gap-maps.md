# Gap Maps: Disciplines by Domain

Use these to identify what the task NEEDS beyond code. Each discipline not covered by an installed skill = ❌ gap.

## Web Project
- **SEO** — invisible to search engines without it
- **Copywriting** — headlines, CTAs, microcopy define conversion
- **Analytics** — can't measure if the web meets its goals
- **Performance** — load speed impacts SEO and user retention
- **Accessibility** — legal requirement in many jurisdictions, expands audience
- **Marketing** — driving traffic to the finished site
- **Legal** — privacy policy, cookie consent, GDPR/CCPA

## Product Launch
- **Pricing strategy** — wrong price kills adoption regardless of quality
- **Copywriting** — landing page copy drives conversion
- **Email marketing** — nurture leads and announce launch
- **Social media** — organic reach and community building
- **A/B testing** — validate assumptions before scaling

## API Development
- **API documentation** — undocumented APIs don't get adopted
- **Security audit** — vulnerabilities in public APIs are critical
- **Rate limiting** — protect from abuse
- **Monitoring/observability** — detect issues before users report them
- **SDK generation** — lower friction for consumers

## Mobile App
- **ASO (App Store Optimization)** — app store visibility
- **Push notifications** — re-engagement strategy
- **Analytics** — usage patterns and retention metrics
- **Crash reporting** — stability is table stakes
- **Deep linking** — user acquisition funnels

## E-commerce
- **Payments** — checkout friction kills sales
- **Tax compliance** — legal requirement, varies by jurisdiction
- **Shipping** — logistics and cost calculation
- **Fraud detection** — chargebacks erode margins
- **Inventory management** — overselling destroys trust

## Content Platform
- **Content strategy** — what to publish, when, for whom
- **Editorial workflow** — review, approval, scheduling
- **Localization** — reach non-English audiences
- **Search optimization** — internal and external discoverability
- **Content analytics** — what performs, what doesn't
